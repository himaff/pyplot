Introduction à pyplot¶

matplotlib.pyplot est une collection de fonctions qui font fonctionner matplotlib comme MATLAB. Chaque fonction pyplot apporte des modifications à une figure : par exemple, crée une figure, crée une zone de traçage dans une figure, trace des lignes dans une zone de traçage, décore le tracé avec des étiquettes, etc.

Dans matplotlib.pyplot, divers états sont conservés à travers les appels de fonction, de sorte qu'il garde une trace d'éléments tels que la figure actuelle et la zone de traçage, et les fonctions de traçage sont dirigées vers les axes actuels (veuillez noter que "axes" ici et dans la plupart des endroits dans la documentation fait référence à la partie *axes* d'une figure <figure_parts> et non au terme mathématique strict pour plus d'un axe).

Note

l'API pyplot est généralement moins flexible que l'API orientée objet. La plupart des appels de fonction que vous voyez ici peuvent également être appelés en tant que méthodes à partir d'un objet ``Axes``. Nous vous recommandons de parcourir les didacticiels et exemples pour voir comment cela fonctionne.

La génération de visualisations avec pyplot est très rapide:

In [1]:
import matplotlib.pyplot as plt
plt.plot([1, 2, 3, 4])
plt.ylabel('Quelques Nombres')
plt.show()

Vous vous demandez peut-être pourquoi l'axe des x va de 0 à 3 et l'axe des y de 1-4. Si vous fournissez une seule liste ou un seul tableau à ~.pyplot.plot, matplotlib suppose qu'il s'agit d'un séquence de valeurs y, et génère automatiquement les valeurs x pour toi. Étant donné que les plages python commencent par 0, le vecteur x par défaut a le même longueur que y mais commence par 0. Par conséquent, les données x sont [0, 1, 2, 3].

~.pyplot.plot est une fonction polyvalente, et prendra un nombre arbitraire de arguments. Par exemple, pour tracer x en fonction de y, vous pouvez écrire :

In [2]:
plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
Out[2]:
[<matplotlib.lines.Line2D at 0x7f5a58cde7c0>]

Mettre en forme le style de votre tracé¶

Pour chaque paire d'arguments x, y, il existe un troisième argument facultatif qui est la chaîne de format qui indique la couleur et le type de ligne de la parcelle. Les lettres et symboles de la chaîne de format proviennent de MATLAB, et vous concaténez une chaîne de couleur avec une chaîne de style de ligne. La chaîne de format par défaut est « b- », qui est une ligne bleue continue. Pour exemple, pour tracer ce qui précède avec des cercles rouges, vous émettez

In [3]:
plt.plot([1, 2, 3, 4], [1, 4, 9, 16], 'ro')
plt.axis([0, 6, 0, 20])
plt.show()

Voir la documentation ~.pyplot.plot pour un liste des styles de ligne et des chaînes de format. le La fonction ~.pyplot.axis dans l'exemple ci-dessus prend un liste de [xmin, xmax, ymin, ymax] et spécifie la fenêtre du haches.

Si matplotlib se limitait à travailler avec des listes, ce serait assez inutile pour le traitement numérique. Généralement, vous utiliserez numpy <http://www.numpy.org>_ tableaux. En fait, toutes les séquences sont converti en tableaux numpy en interne. L'exemple ci-dessous illustre tracer plusieurs lignes avec des styles de format différents dans un seul appel de fonction en utilisant des tableaux.

In [4]:
import numpy as np

# temps d'échantillonnage uniforme à des intervalles de 200 ms
t = np.arange(0., 5., 0.2)

# tirets rouges, carrés bleus et triangles verts
plt.plot(t, t, 'r--', t, t**2, 'bs', t, t**3, 'g^')
plt.show()

Tracé avec des chaînes de mots-clés¶

Il y a des cas où vous avez des données dans un format qui vous permet accéder à des variables particulières avec des chaînes. Par exemple, avec numpy.recarray ou pandas.DataFrame.

Matplotlib vous permet de fournir un tel objet avec l'argument du mot clé data. Si fourni, vous pouvez générer des graphiques avec les chaînes correspondant à ces variables.

In [5]:
data = {'a': np.arange(50),
        'c': np.random.randint(0, 50, 50),
        'd': np.random.randn(50)}
data['b'] = data['a'] + 10 * np.random.randn(50)
data['d'] = np.abs(data['d']) * 100

plt.scatter('a', 'b', c='c', s='d', data=data)
plt.xlabel('entrée a')
plt.ylabel('entrée b')
plt.show()

Tracé avec des variables catégorielles¶

Il est également possible de créer un graphique à l'aide de variables catégorielles. Matplotlib vous permet de passer directement des variables catégorielles à de nombreuses fonctions de tracé. Par example:

In [6]:
names = ['group_a', 'group_b', 'group_c']
values = [1, 10, 100]

plt.figure(figsize=(9, 3))

plt.subplot(131)
plt.bar(names, values)
plt.subplot(132)
plt.scatter(names, values)
plt.subplot(133)
plt.plot(names, values)
plt.suptitle('Tracé catégoriel')
plt.show()

Contrôler les propriétés de la ligne¶

Les lignes ont de nombreux attributs que vous pouvez définir : largeur de ligne, style de tiret, anticrénelé, etc. voir matplotlib.lines.Line2D. Il y a plusieurs façons de définir les propriétés de la ligne

  • Utilisez des arguments de mots clés : :

    plt.plot(x, y, largeur de ligne=2.0)

  • Utiliser les méthodes setter d'une instance Line2D. plot renvoie une liste d'objets Line2D ; par exemple, ligne1, ligne2 = plot(x1, y1, x2, y2). Dans le code ci-dessous, nous supposerons que nous n'avons que une ligne pour que la liste renvoyée soit de longueur 1. Nous utilisons le déballage des tuples avec line, pour obtenir le premier élément de cette liste : :

    ligne, = plt.plot(x, y, '-')
line.set_antialiased(False) # désactiver l'anticrénelage

  • Utilisez ~.pyplot.setp. L'exemple ci-dessous utilise une fonction de style MATLAB pour définir plusieurs propriétés sur une liste de lignes. setp fonctionne de manière transparente avec une liste d'objets ou un seul objet. Vous pouvez soit utiliser des arguments de mot-clé python ou Paires chaîne/valeur de style MATLAB : :

    lignes = plt.plot(x1, y1, x2, y2)
# utiliser des arguments de mot-clé
plt.setp(lignes, couleur='r', largeur de ligne=2.0)
# ou paires de valeurs de chaîne de style MATLAB
plt.setp(lignes, 'couleur', 'r', 'largeur de ligne', 2.0)

Travailler avec plusieurs figures et axes¶

MATLAB et .pyplot, ont le concept de la figure actuelle et les axes actuels. Toutes les fonctions de traçage s'appliquent au courant haches. La fonction ~.pyplot.gca renvoie les axes courants (un instance matplotlib.axes.Axes), et ~.pyplot.gcf renvoie l'instance actuelle figure (une instance matplotlib.figure.Figure). Normalement, vous n'avez pas à s'inquiéter à ce sujet, parce que tout est pris en charge dans les coulisses. Au dessous de est un script pour créer deux intrigues secondaires.

In [7]:
def f(t):
    return np.exp(-t) * np.cos(2*np.pi*t)

t1 = np.arange(0.0, 5.0, 0.1)
t2 = np.arange(0.0, 5.0, 0.02)

plt.figure()
plt.subplot(211)
plt.plot(t1, f(t1), 'bo', t2, f(t2), 'k')

plt.subplot(212)
plt.plot(t2, np.cos(2*np.pi*t2), 'r--')
plt.show()

L'appel ~.pyplot.figure ici est facultatif car une figure sera créée s'il n'en existe pas, tout comme un axe sera créé (équivalent à un subplot()) s'il n'en existe pas. L'appel ~.pyplot.subplot spécifie numrows, numcols, plot_number où plot_number va de 1 à numrows*numcols. Les virgules dans l'appel subplot sont facultatif si numrows*numcols<10. Donc subplot(211) est identique à sous-intrigue(2, 1, 1).

Vous pouvez créer un nombre arbitraire de sous-parcelles et haches. Si vous souhaitez placer un axe manuellement, c'est-à-dire pas sur un grille rectangulaire, utilisez ~.pyplot.axes, qui vous permet de spécifier l'emplacement comme axes([gauche, bas, largeur, hauteur]) où toutes les valeurs sont en fraction (0 à 1) coordonnées. Voir /gallery/subplots_axes_and_figures/axes_demo pour un exemple de placer des axes manuellement et /gallery/subplots_axes_and_figures/subplot_demo pour un exemple avec beaucoup de sous-parcelles.

Vous pouvez créer plusieurs figures en utilisant plusieurs ~.pyplot.figure appels avec un chiffre croissant numéro. Bien entendu, chaque figure peut contenir autant d'axes et de sous-parcelles comme ton coeur le désire

import matplotlib.pyplot as plt
    plt.figure(1)                # the first figure
    plt.subplot(211)             # the first subplot in the first figure
    plt.plot([1, 2, 3])
    plt.subplot(212)             # the second subplot in the first figure
    plt.plot([4, 5, 6])


    plt.figure(2)                # a second figure
    plt.plot([4, 5, 6])          # creates a subplot() by default

    plt.figure(1)                # figure 1 current; subplot(212) still current
    plt.subplot(211)             # make subplot(211) in figure1 current
    plt.title('Easy as 1, 2, 3') # subplot 211 title

Vous pouvez effacer le chiffre actuel avec ~.pyplot.clf et les axes courants avec ~.pyplot.cla. Si tu trouves c'est ennuyeux que les états (en particulier l'image actuelle, la figure et les axes) sont entretenus pour vous en coulisses, ne désespérez pas : ce n'est qu'un mince wrapper avec état autour d'une API orientée objet, que vous pouvez utiliser à la place (voir /tutorials/intermediate/artists)

Si vous faites beaucoup de chiffres, vous devez en connaître un plus de chose : la mémoire nécessaire pour une figure n'est pas complètement libéré jusqu'à ce que la figure soit explicitement fermée avec ~.pyplot.close. Supprimer toutes les références au figure, et/ou en utilisant le gestionnaire de fenêtres pour tuer la fenêtre dans laquelle le chiffre apparaît à l'écran, ne suffit pas, car pyplot maintient les références internes jusqu'à ~.pyplot.close est appelé.

Travailler avec du texte¶

~.pyplot.text peut être utilisé pour ajouter du texte dans un emplacement arbitraire, et ~.pyplot.xlabel, ~.pyplot.ylabel et ~.pyplot.title sont utilisés pour ajouter texte aux emplacements indiqués (voir /tutorials/text/text_intro pour un exemple plus détaillé)

In [8]:
mu, sigma = 100, 15
x = mu + sigma * np.random.randn(10000)

# l' histogramme de données
n, bins, patches = plt.hist(x, 50, density=1, facecolor='g', alpha=0.75)


plt.xlabel('Intelligents')
plt.ylabel('Probabilité')
plt.title('Histogramme de QI')
plt.text(60, .025, r'$\mu=100,\ \sigma=15$')
plt.axis([40, 160, 0, 0.03])
plt.grid(True)
plt.show()

Toutes les fonctions ~.pyplot.text renvoient un matplotlib.text.Text exemple. Tout comme pour les lignes ci-dessus, vous pouvez personnaliser les propriétés en en passant des arguments de mot-clé dans les fonctions de texte ou en utilisant ~.pyplot.setp ::

t = plt.xlabel('mes données', taille de police=14, couleur='rouge')

Ces propriétés sont décrites plus en détail dans /tutorials/text/text_props.

Utiliser des expressions mathématiques dans le texte¶

matplotlib accepte les expressions d'équation TeX dans n'importe quelle expression de texte. Par exemple pour écrire l'expression $\sigma_i=15$ dans le titre, vous pouvez écrire une expression TeX entourée de signes dollar : :

plt.title(r'$\sigma_i=15$')

Le r précédant la chaîne du titre est important -- il signifie que la chaîne est une chaîne brute et de ne pas traiter les barres obliques inverses comme python s'échappe. matplotlib a un analyseur d'expression TeX intégré et moteur de mise en page et fournit ses propres polices mathématiques -- pour plus de détails, voir /tutorials/text/mathtext. Ainsi, vous pouvez utiliser du texte mathématique sur toutes les plateformes sans nécessiter l'installation de TeX. Pour ceux qui ont LaTeX et dvipng installé, vous pouvez également utiliser LaTeX pour formater votre texte et incorporer la sortie directement dans vos chiffres d'affichage ou enregistrés postscript -- voir /tutorials/text/usetex.

Texte d'annotation¶

Les utilisations de la fonction de base ~.pyplot.text ci-dessus placer le texte à une position arbitraire sur les axes. Une utilisation courante pour le texte est d'annoter une caractéristique de l'intrigue, et le La méthode ~.pyplot.annotate fournit une aide fonctionnalité pour faciliter les annotations. Dans une annotation, il y a deux points à considérer : l'emplacement étant annoté représenté par l'argument xy et l'emplacement du texte xytext. Les deux ces arguments sont des tuples (x, y).

In [9]:
ax = plt.subplot()

t = np.arange(0.0, 5.0, 0.01)
s = np.cos(2*np.pi*t)
line, = plt.plot(t, s, lw=2)

plt.annotate('max local', xy=(2, 1), xytext=(3, 1.5),
             arrowprops=dict(facecolor='black', shrink=0.05),
             )

plt.ylim(-2, 2)
plt.show()

Dans cet exemple de base, le xy (pointe de flèche) et le xytext les emplacements (emplacement du texte) sont en coordonnées de données. Il y a un variété d'autres systèmes de coordonnées que l'on peut choisir -- voir annotations-tutorial et plotting-guide-annotation pour des détails. Vous trouverez d'autres exemples dans :doc:/gallery/text_labels_and_annotations/annotation_demo.

Axes logarithmiques et autres axes non linéaires¶

:mod:matplotlib.pyplot prend en charge non seulement les échelles d'axe linéaire, mais aussi échelles logarithmique et logit. Ceci est couramment utilisé si les données s'étendent sur de nombreuses commandes de grandeur. Changer l'échelle d'un axe est simple :

plt.xscale('log')

Un exemple de quatre tracés avec les mêmes données et des échelles différentes pour l'axe des y est montré ci-dessous.

In [10]:
# Correction de l'état aléatoire pour la reproductibilité
np.random.seed(19680801)

# composer des données dans l'intervalle ouvert (0, 1)
y = np.random.normal(loc=0.5, scale=0.4, size=1000)
y = y[(y > 0) & (y < 1)]
y.sort()
x = np.arange(len(y))

# tracé avec différentes échelles d'axes
plt.figure()

# lineaire
plt.subplot(221)
plt.plot(x, y)
plt.yscale('linear')
plt.title('linear')
plt.grid(True)

# log
plt.subplot(222)
plt.plot(x, y)
plt.yscale('log')
plt.title('log')
plt.grid(True)

# symmetric log
plt.subplot(223)
plt.plot(x, y - y.mean())
plt.yscale('symlog', linthresh=0.01)
plt.title('symlog')
plt.grid(True)

# logit
plt.subplot(224)
plt.plot(x, y)
plt.yscale('logit')
plt.title('logit')
plt.grid(True)

# Ajustez la disposition de la sous-intrigue, car la logit peut prendre plus de place
# que d'habitude, en raison des étiquettes y-tick comme "1 - 10^{-3}"
plt.subplots_adjust(top=0.92, bottom=0.08, left=0.10, right=0.95, hspace=0.25,
                    wspace=0.35)

plt.show()

Il est également possible d'ajouter votre propre échelle, voir adding-new-scales pour des détails.
