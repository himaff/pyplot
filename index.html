Introduction à pyplot¶

matplotlib.pyplot est une collection de fonctions qui font fonctionner matplotlib comme MATLAB. Chaque fonction pyplot apporte des modifications une figure: par exemple, cre une figure, cre une zone de traage dans une figure, trace des lignes dans une zone de traage, dcore le trac avec des tiquettes, etc.

Dans matplotlib.pyplot, divers tats sont conservs travers les appels de fonction, de sorte qu'il garde une trace d'lments tels que la figure actuelle et la zone de traage, et les fonctions de traage sont diriges vers les axes actuels (veuillez noter que "axes" ici et dans la plupart des endroits dans la documentation fait rfrence la partie *axes* d'une figure <figure_parts> et non au terme mathmatique strict pour plus d'un axe).

Pour chaque paire d'arguments x, y, il existe un troisime argument facultatif qui est la chane de format qui indique la couleur et le type de ligne de la parcelle. Travailler avec plusieurs figures et axes¶

MATLAB et .pyplot, ont le concept de la figure actuelle et les axes actuels. Toutes les fonctions de traage s'appliquent au courant haches. La fonction ~.pyplot.gca renvoie les axes courants (un instance matplotlib.axes.Axes), et ~.pyplot.gcf renvoie l'instance actuelle figure (une instance matplotlib.figure.Figure). Normalement, vous n'avez pas s'inquiter ce sujet, parce que tout est pris en charge dans les coulisses. Au dessous de est un script pour crer deux intrigues secondaires.

In [7]:
def f(t):
    return np.exp(-t) * np.cos(2*np.pi*t)

t1 = np.arange(0.0, 5.0, 0.1)
t2 = np.arange(0.0, 5.0, 0.02)

plt.figure()
plt.subplot(211)
plt.plot(t1, f(t1), 'bo', t2, f(t2), 'k')

plt.subplot(212)
plt.plot(t2, np.cos(2*np.pi*t2), 'r--')
plt.show()

L'appel ~.pyplot.figure ici est facultatif car une figure sera cre s'il n'en existe pas, tout comme un axe sera cr (quivalent un subplot()) s'il n'en existe pas. L'appel ~.pyplot.subplot spcifie numrows, numcols, plot_number o plot_number va de 1 numrows*numcols. Les virgules dans l'appel subplot sont facultatif si numrows*numcols<10. Donc subplot(211) est identique sous-intrigue(2, 1, 1).

Vous pouvez crer un nombre arbitraire de sous-parcelles et haches. Si vous souhaitez placer un axe manuellement, c'est--dire pas sur un grille rectangulaire, utilisez ~.pyplot.axes, qui vous permet de spcifier l'emplacement comme axes([gauche, bas, largeur, hauteur]) o toutes les valeurs sont en fraction (0 1) coordonnes. Voir /gallery/subplots_axes_and_figures/axes_demo pour un exemple de placer des axes manuellement et /gallery/subplots_axes_and_figures/subplot_demo pour un exemple avec beaucoup de sous-parcelles.

Vous pouvez crer plusieurs figures en utilisant plusieurs ~.pyplot.figure appels avec un chiffre croissant numro. Bien entendu, chaque figure peut contenir autant d'axes et de sous-parcelles comme ton coeur le dsire

import matplotlib.pyplot as plt
    plt.figure(1)                # the first figure
    plt.subplot(211)             # the first subplot in the first figure
    plt.plot([1, 2, 3])
    plt.subplot(212)             # the second subplot in the first figure
    plt.plot([4, 5, 6])


    plt.figure(2)                # a second figure
    plt.plot([4, 5, 6])          # creates a subplot() by default

    plt.figure(1)                # figure 1 current; subplot(212) still current
    plt.subplot(211)             # make subplot(211) in figure1 current
    plt.title('Easy as 1, 2, 3') # subplot 211 title

Vous pouvez effacer le chiffre actuel avec ~.pyplot.clf et les axes courants avec ~.pyplot.cla. Si tu trouves c'est ennuyeux que les tats (en particulier l'image actuelle, la figure et les axes) sont entretenus pour vous en coulisses, ne dsesprez pas : ce n'est qu'un mince wrapper avec tat autour d'une API oriente objet, que vous pouvez utiliser la place (voir /tutorials/intermediate/artists)

Supprimer toutes les rfrences au figure, et/ou en utilisant le gestionnaire de fentres pour tuer la fentre dans laquelle le chiffre apparat l'cran, ne suffit pas, car pyplot maintient les rfrences internes jusqu' ~.pyplot.close est appel.

matplotlib accepte les expressions d'quation TeX dans n'importe quelle expression de texte. Par exemple pour crire l'expression $\sigma_i=15$ dans le titre, vous pouvez crire une expression TeX entoure de signes dollar::

Le r prcdant la chane du titre est important -- il signifie que la chane est une chane brute et de ne pas traiter les barres obliques inverses comme python s'chappe. matplotlib a un analyseur d'expression TeX intgr et moteur de mise en page et fournit ses propres polices mathmatiques -- pour plus de dtails, voir /tutorials/text/mathtext. Ainsi, vous pouvez utiliser du texte mathmatique sur toutes les plateformes sans ncessiter l'installation de TeX. Pour ceux qui ont LaTeX et dvipng install, vous pouvez galement utiliser LaTeX pour formater votre texte et incorporer la sortie directement dans vos chiffres d'affichage ou enregistrs postscript -- voir /tutorials/text/usetex.

Texte d'annotation¶

Les utilisations de la fonction de base ~.pyplot.text ci-dessus placer le texte une position arbitraire sur les axes. Une utilisation courante pour le texte est d'annoter une caractristique de l'intrigue, et le La mthode ~.pyplot.annotate fournit une aide fonctionnalit pour faciliter les annotations. Dans une annotation, il y a deux points considrer : l'emplacement tant annot reprsent par l'argument xy et l'emplacement du texte xytext. Les deux ces arguments sont des tuples (x, y).

Dans cet exemple de base, le xy (pointe de flche) et le xytext les emplacements (emplacement du texte) sont en coordonnes de donnes. Axes logarithmiques et autres axes non linéaires¶

:mod:matplotlib.pyplot prend en charge non seulement les chelles d'axe linaire, mais aussi chelles logarithmique et logit. Ceci est couramment utilis si les donnes s'tendent sur de nombreuses commandes de grandeur. Changer l'chelle d'un axe est simple :
